# Sales Data Analysis and Reporting

## Overview
This project analyzes retail sales transaction data to identify trends and generate business insights using Python, SQL, and Excel.

## Business Objective
To understand sales performance, identify yearly and monthly trends, and present results through structured reports and dashboards.

## Tools Used
- Python (Pandas)
- SQL (SQLite)
- Excel
- Jupyter Notebook

## Key Analysis
- Data cleaning and preprocessing
- Year-wise and month-wise sales analysis
- SQL-based aggregation
- Excel reporting and dashboard creation

## Files Included
- analysis.ipynb – Complete code
- sales_report.xlsx – Reports and dashboard
- Project PPT – Presentation
- Project Report – Detailed documentation

## Conclusion
The project demonstrates how Python, SQL, and Excel can be combined to solve real-world retail data analysis problems.
